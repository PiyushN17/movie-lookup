This project allows users to search for information about movies and TV shows using the OMDb (Open Movie Database) API. By entering the name of a movie or TV show into the search bar, users can retrieve detailed information, including the title, year of release, genre, IMDb rating, director, actors, plot summary, box office performance, and more. Additionally, users can view the movie or show poster, if available.

The application provides a clean and interactive user interface built with HTML, CSS, and JavaScript. It includes functionality for handling search errors, resetting the input, and displaying relevant movie details.

Features

Search functionality: Enter a movie or TV show name to retrieve detailed information such as:

Title, Year, Type (movie/TV show)

Genre, IMDb Rating, Age Rating, Release Date

Runtime, Director, Actors, Awards, Box Office

Plot summary and poster image (if available)

Error handling: If a movie or TV show is not found or there is an issue with the search, an error message is displayed.

Reset functionality: Reset the displayed information to its default state with a single button click.

Responsive design: The layout adjusts to different screen sizes, ensuring a user-friendly experience across devices, including desktops and mobile phones.

Technologies Used

HTML5: Structure and content of the webpage.

CSS3: Styling and layout design with a focus on responsiveness.

JavaScript: Interactivity, including API calls to the OMDb API, search functionality, and dynamic content updates.

How to Use

Search for a Movie or TV Show: Enter a title in the search input field and click the Search button. Information about the movie or show will be displayed below the input field.

Reset: To clear the displayed information and disable the reset button, click the Reset button.

Error Handling: If no results are found or the input is empty, an error message will appear.
