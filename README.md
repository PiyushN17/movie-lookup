This project allows users to search for information about movies and TV shows using the OMDb (Open Movie Database) API. By entering the name of a movie or TV show into the search bar, users can retrieve detailed information, including the title, year of release, genre, IMDb rating, director, actors, plot summary, box office performance, and more. Additionally, users can view the movie or show poster, if available.

The application provides a clean, interactive user interface built with HTML, CSS, and JavaScript, offering seamless functionality for searching, error handling, and displaying movie details.

Features

Search Functionality: Enter a movie or TV show name to retrieve detailed information such as:

Title, Year, Type (movie/TV show)

Genre, IMDb Rating, Age Rating, Release Date

Runtime, Director, Actors, Awards, Box Office

Plot summary and poster image (if available)

Error Handling: If a movie or TV show is not found, or there is an issue with the search, an error message will be displayed.

Reset Functionality: Reset the displayed information to its default state with a single click.

Responsive Design: The layout adjusts to different screen sizes, ensuring a user-friendly experience on both desktops and mobile devices.

**Technologies Used**

 - HTML5: Provides the structure and content of the webpage.

 - CSS3: Responsible for styling and layout design, with a focus on responsiveness.

 - JavaScript: Adds interactivity, including API calls to the OMDb API, search functionality, and dynamic content updates.

**How to Use**

Search for a Movie or TV Show:

Enter a title in the search input field and click the "Search" button. Information about the movie or show will be displayed below the input field.

Reset:

To clear the displayed information, click the "Reset" button.

Error Handling:

If no results are found or the input is empty, an error message will appear.
